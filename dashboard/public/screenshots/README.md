# Screenshots

This directory contains screenshots of the dashboard.

## How to Generate Screenshots

1. Start the development server:
   ```bash
   npm run dev
   ```

2. Open the dashboard in your browser:
   - Homepage: http://localhost:3000
   - Tests Page: http://localhost:3000/tests
   - Parity Page: http://localhost:3000/parity
   - Agents Page: http://localhost:3000/agents

3. Take screenshots using:
   - macOS: `Cmd + Shift + 4`
   - Browser DevTools: Device toolbar for mobile views

4. Save screenshots in this directory:
   - `homepage-desktop.png` - Desktop view of homepage
   - `homepage-mobile.png` - Mobile view of homepage
   - `tests-page.png` - Tests page
   - `parity-page.png` - Parity page
   - `agents-page.png` - Agents page
   - `preview.png` - Hero image for README

## Recommended Sizes

- Desktop: 1440x900
- Mobile: 375x667 (iPhone SE)
- Tablet: 768x1024 (iPad)
