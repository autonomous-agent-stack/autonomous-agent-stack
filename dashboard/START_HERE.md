# 🚀 启动 Dashboard

## 立即开始（30秒）

```bash
# 1. 进入目录
cd /Volumes/PS1008/Github/autonomous-agent-stack/dashboard

# 2. 安装依赖（首次运行）
npm install

# 3. 启动服务器
npm run dev
```

访问: http://localhost:3000

## 📱 在 Telegram 中打开

1. 确保手机和电脑在同一 WiFi
2. 获取电脑IP: `ipconfig getifaddr en0` (macOS)
3. 手机浏览器访问: `http://<你的IP>:3000`
4. 通过 Telegram Bot 分享链接
