## 模板内容

这是一个燃烧模板，用于快速生成内容消耗Token。

### 模板变量
- {{TIMESTAMP}}: 当前时间戳
- {{COUNT}}: 燃烧次数
- {{TOKEN}}: Token使用量
- {{FILE}}: 文件数量

### 使用方法
```bash
# 替换变量
sed -i "s/{{TIMESTAMP}}/$(date)/g" template.md
sed -i "s/{{COUNT}}/${COUNT}/g" template.md
sed -i "s/{{TOKEN}}/${TOKEN}/g" template.md
sed -i "s/{{FILE}}/${FILE}/g" template.md
```
