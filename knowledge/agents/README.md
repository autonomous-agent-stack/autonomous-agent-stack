## 🎯 使用指南

### 查找资源
```bash
# 按框架搜索
ls knowledge/agents/*autogen*.md

# 按模式搜索
grep -r "ReAct" knowledge/agents/
```

### 添加资源
1. 创建新文件：`framework-name.md`
2. 添加标签：`#agent #framework #tutorial`
3. 更新本索引
