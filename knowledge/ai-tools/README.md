## 📖 使用指南

### 如何选择 AI 工具？

**评估维度**：
1. **成本** - API 费用 vs 自部署
2. **性能** - 响应时间 + 准确率
3. **易用性** - 学习曲线
4. **集成** - API + SDK
5. **隐私** - 数据安全

### 决策矩阵

| 场景 | 推荐工具 | 理由 |
|------|---------|------|
| 个人学习 | ChatGPT/Claude | 免费额度足够 |
| 编程辅助 | Cursor/Claude Code | IDE 集成 |
| 图像生成 | Midjourney/DALL-E | 质量最佳 |
| 企业部署 | 私有化模型 | 数据安全 |
| 研究 | 开源模型 | 可定制性 |

**测试任务**：代码审查 + Bug 修复
