## 💡 核心概念

### 多模态 = 跨模态理解 + 融合生成
```
文本 + 图像 + 音频 → 统一表示 → 理解/生成
    ↑___________________|
         交互增强
```

### 多模态架构
```
编码器（文本/图像/音频）
    ↓
特征提取
    ↓
跨模态对齐
    ↓
融合表示
    ↓
解码器（生成/理解）
```

### 1. 视觉问答
