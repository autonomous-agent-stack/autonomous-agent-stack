## 🚀 AI 学习路径

### 路径 1：AI Agent 开发者
**时长**：3 个月
**前置知识**：Python 基础

```
第 1 个月：基础
├─ Week 1-2: LLM 基础 + Prompt Engineering
├─ Week 3-4: Function Calling + 工具调用
└─ 项目：构建简单问答机器人

第 2 个月：进阶
├─ Week 5-6: RAG 系统 + 向量数据库
├─ Week 7-8: Agent 框架（LangChain/OpenClaw）
└─ 项目：构建知识库助手

第 3 个月：高级
├─ Week 9-10: 多智能体系统
├─ Week 11-12: 生产部署 + 监控
└─ 项目：构建团队协作 Agent
```

**详细指南**：[ai-agent-developer-path.md](./ai-agent-developer-path.md)

## 📊 学习方法

### 费曼学习法
1. **选择概念** - 确定学习主题
2. **教授他人** - 用简单语言解释
3. **发现盲点** - 识别理解不足
4. **回顾简化** - 巩固知识

### 刻意练习
- ✅ 明确目标
- ✅ 专注练习
- ✅ 即时反馈
- ✅ 逐步提升

### 项目驱动学习
- 🎯 实际项目 > 理论学习
- 🔄 迭代改进
- 📝 记录过程
- 🤝 获取反馈
