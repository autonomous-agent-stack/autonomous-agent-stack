## 🏗️ 技术架构

### Agent 类型
- **协调器 (Coordinator)**: 任务分配与调度
- **执行器 (Executor)**: 具体任务执行
- **监控器 (Monitor)**: 状态监控与日志
- **优化器 (Optimizer)**: 性能优化与调优

### 通信机制
- **消息传递**: 异步消息队列
- **事件总线**: 发布-订阅模式
- **RPC 调用**: 同步通信支持

### GLM-5 集成
- **模型适配**: API 接口统一
- **性能优化**: 推理加速
- **容错机制**: 降级策略
