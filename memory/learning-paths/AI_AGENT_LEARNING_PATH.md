## 🎯 Week 1-2: 基础知识

### 必学内容

1. **Python 进阶**（20 小时）

   - 类型注解（typing）
   - 装饰器和高阶函数
   - 测试框架（pytest）

2. **API 基础**（10 小时）
   - REST API 设计
   - FastAPI 框架
   - API 认证和授权
   - 错误处理

3. **LLM 基础**（10 小时）
   - Transformer 架构
   - Attention 机制
   - Prompt Engineering
   - API 调用（OpenAI/Claude/GLM）

### 实践项目

- ✅ 构建一个简单的 API 服务

- ✅ 实现 Prompt 模板系统

## 💡 学习建议

### ✅ 有效学习方法

1. **项目驱动学习** - 通过实际项目学习
2. **每日练习** - 每天至少 1-2 小时
3. **社区参与** - 加入 AI Agent 社区
4. **文档优先** - 先看官方文档
5. **代码审查** - 参与开源项目

### ❌ 避免的错误

1. **只看不练** - 必须动手实践
2. **追求完美** - 先完成再优化
3. **孤军奋战** - 加入社区交流
4. **跳过基础** - 基础不牢地动山摇
5. **过早优化** - 先让它工作，再让它快
