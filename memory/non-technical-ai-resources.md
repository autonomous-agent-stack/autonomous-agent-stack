## 🎯 场景化推荐

### ✍️ 写作场景

#### 适合工具
- **Claude**: 长文章、分析、润色
- **ChatGPT**: 创意写作、头脑风暴
- **Jasper**: 营销文案（付费）
- **Copy.ai**: 广告文案（付费）

#### Prompt 模板

**文章写作**
```
请帮我写一篇关于 [主题] 的文章。

要求：
- 风格：[专业/轻松/学术/幽默]
- 字数：[1000/2000] 字左右
- 目标读者：[学生/专业人士/普通大众]
- 重点内容：[要点1, 要点2, 要点3]

请先列出大纲，然后分段撰写。
```

**邮件撰写**
```
请帮我写一封邮件。

收件人：[客户/老板/同事]
主题：[请假/汇报/邀请]
语调：[正式/友好/紧急]
关键信息：[要点]

邮件不要太长，3-4 段即可。
```

**内容润色**
```
请帮我润色以下内容：

[粘贴内容]

要求：
- 保持原意
- 更流畅自然
- 修正语法错误
- 优化表达
```

### 💻 编程学习场景

#### 适合工具
- **ChatGPT**: 代码解释、调试
- **Claude**: 代码审查、重构建议
- **GitHub Copilot**: 代码补全（IDE 集成）
- **Cursor**: AI 编程编辑器

#### Prompt 模板

**概念学习**
```
我想学习 [编程概念/语言]。

请：
1. 用简单的话解释这个概念
2. 举一个生活中的类比
3. 给一个最简单的代码示例
4. 告诉我常见错误

我完全没有编程基础。
```

**代码调试**
```
这段代码报错了，请帮我找出问题：

[粘贴代码]

错误信息：[错误信息]

请：
1. 解释错误原因
2. 提供修复方案
3. 告诉我如何避免这类错误
```

### 📚 学习场景

#### Prompt 模板

**概念理解**
```
请用最简单的话解释 [概念]。

要求：
1. 像对 10 岁小孩解释一样
2. 举一个生活中的例子
3. 说明它为什么重要
4. 如何应用到实际
```

**学习计划**
```
我想学习 [主题]。

情况：
- 我的基础：[完全零基础/有一些基础]
- 每天可用时间：[1 小时/2 小时]
- 学习目标：[了解基本概念/能够实际应用]
- 时间周期：[1 个月/3 个月]

请制定一个详细的学习计划，包括：
1. 学习路径
2. 推荐资源
3. 每周任务
4. 检验标准
```

**知识测试**
```
我刚学习了 [主题]，请测试我的理解：

1. 出 5 道选择题（由浅入深）
2. 出 2 道简答题
3. 出 1 道应用题

等我回答后，请：
- 给出分数
- 指出错误
- 提供正确答案和解释
```

## ⚠️ 注意事项

### 🔒 隐私与安全

1. **不要输入敏感信息**
   - 密码、身份证号、银行卡号
   - 家庭住址、电话号码
   - 公司机密信息

2. **数据使用政策**
   - 了解工具的数据使用政策
   - 区分训练模式和隐私模式
   - 企业用户注意合规

### ✅ 事实核查

1. **AI 可能产生错误**
   - 幻觉（Hallucination）：编造不存在的信息
   - 知识过时：训练数据有截止日期
   - 偏见：训练数据中的偏见

2. **验证方法**
   - 交叉验证多个来源
   - 检查引用和来源
   - 对关键信息进行确认

### ⚖️ 伦理与版权
