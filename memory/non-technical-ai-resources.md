## 🎯 场景化 Prompt 模板

### 写作类
| 场景 | Prompt 示例 |
|------|-------------|
| 文章写作 | "请帮我写一篇关于[主题]的文章，风格要[专业/轻松/学术]，目标读者是[人群]。" |
| 邮件撰写 | "请帮我写一封[正式/友好]的邮件，主题是[xxx]，收件人是[xxx]。" |
| 总结提炼 | "请用3个要点总结以下内容：[粘贴内容]" |

### 学习类
| 场景 | Prompt 示例 |
|------|-------------|
| 概念解释 | "请用简单的话解释[概念]，就像对10岁小孩解释一样。" |
| 学习计划 | "我想学习[主题]，请帮我制定一个[30天/3个月]的学习计划。" |
| 知识测试 | "请出5道关于[主题]的选择题，测试我的理解。" |

### 工作类
| 场景 | Prompt 示例 |
|------|-------------|
| 会议总结 | "请总结以下会议记录的关键点：[粘贴内容]" |
| 头脑风暴 | "请帮我想10个关于[主题]的创意想法。" |
| 翻译润色 | "请将以下内容翻译成[语言]，并保持[正式/轻松]的语调：[内容]" |

## 📖 推荐学习路径

### 路径 1：完全零基础（1-2周）
1. 注册 Claude/ChatGPT 账号
2. 阅读 Prompt Engineering Guide（基础部分）
3. 尝试 Awesome ChatGPT Prompts 中的模板
4. 完成 Claude 101 课程

### 路径 2：进阶学习（2-4周）
1. 完成 AI Fluency 课程
2. 学习思维链（Chain of Thought）技术
3. 实践 RAG 概念
4. 探索多模态应用

### 路径 3：专业应用（持续）
1. 根据工作场景定制 Prompt
2. 建立个人 Prompt 库
3. 关注 AI 工具更新
4. 参与社区交流

## ⚠️ 注意事项

1. **隐私保护**: 不要输入敏感信息（密码、地址、身份证等）
2. **事实核查**: AI 可能产生错误，重要信息需要验证
3. **版权意识**: 生成内容可能涉及版权问题
4. **适度使用**: AI 是工具，不是替代思考

## 📌 快速开始

**第一步**: 打开 https://claude.ai 或 https://chat.openai.com/

**第二步**: 输入你的第一个 Prompt：
```
你好！我是 AI 新手。请用简单的话告诉我：
1. 你能做什么？
2. 我该怎么向你提问？
3. 有什么注意事项？
```

**第三步**: 开始对话！

---

> 💡 **提示**: 最好的学习方式是**多用**。每天花 10 分钟与 AI 对话，一周后你就会掌握基本技巧。
