## 🔗 快速开始

### SuperAGI（5 分钟）

```bash
# 克隆仓库
git clone https://github.com/TransformerOptimus/SuperAGI.git

# 安装依赖
cd SuperAGI
pip install -r requirements.txt

# 配置环境
cp .env.example .env
# 编辑 .env 填入 API keys

# 启动
python main.py
```

### voltagent（5 分钟）

```bash
# 克隆仓库
git clone https://github.com/VoltAgent/voltagent.git

# 安装依赖
cd voltagent
npm install

# 配置环境
cp .env.example .env
# 编辑 .env 填入 API keys

# 启动
npm start
```
