# 2026-04-23

- Telegram `/status` 现在对本地 `127.0.0.1` / 私有面板地址不再直接回传长 token 链接；若配置了 Mini App，则优先提示使用按钮打开。
- Mac worker 注册能力补上了 `claude_runtime`，使其在 worker 盘点里更准确地表达“可代理 Hermes runtime lane”。
- 当前 AAS 接入已有 Hermes 的最佳实践已经固定为：保留 `hermes` CLI 在 `PATH` 中，启动 AAS API + Telegram poller + Mac worker，由 Mac worker 注册/心跳并通过 `WorkerRuntimeDispatchService` 调 Hermes；单独运行 `hermes gateway` 不会自动进入 AAS worker registry。
