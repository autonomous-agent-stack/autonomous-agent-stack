## 🔥 核心要点总结

### ✅ 必须做

1. **开启 VPN**
   - 推荐节点: 美国（美西/美东）
   - 测试: 访问 google.com

2. **浏览器语言设为英语**
   - Chrome 设置 → 语言 → English (United States)
   - 移到顶部 → 重启浏览器

3. **使用 Gmail 登录**
   - Gemini 需要 Google 账号
   - 手机号验证（可选）

### ❌ 不要做

1. **不要使用中文浏览器界面**
   - 会导致区域限制

2. **不要关闭 VPN**
   - 使用期间全程开启

3. **不要使用中国大陆手机号**
   - 验证时选择"跳过"
