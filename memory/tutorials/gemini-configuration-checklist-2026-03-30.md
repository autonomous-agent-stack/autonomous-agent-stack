## 📋 配置前准备

### 1. 网络环境检查

- [ ] **VPN 已安装并激活**
  - 推荐: ExpressVPN / NordVPN / Clash
  - 测试: 访问 https://www.google.com
  - 要求: 页面正常加载

- [ ] **选择合适的节点**
  - 首选: 美国（美西/美东）
  - 备选: 日本 / 新加坡 / 英国
  - 避免: 香港 / 澳门 / 中国大陆

- [ ] **网络速度测试**
  - 访问: https://speedtest.net
  - 要求: 下载 > 5 Mbps
  - 延迟: < 200ms

---

### 2. 浏览器配置检查

- [ ] **使用 Chrome 浏览器**
  - 版本: 最新版（120+）
  - 检查: chrome://settings/help
  - 更新: 如有更新，立即安装

- [ ] **语言设置为英语**
  ```
  检查步骤:
  1. 打开 chrome://settings/languages
  2. 首选语言应为: English (United States)
  3. 如不是，点击 ⋮ → 移到顶部
  4. 重启浏览器
  ```

- [ ] **清除旧数据**
  ```
  操作步骤:
  1. chrome://settings/clearBrowserData
  2. 时间范围: 过去 1 小时
  3. 勾选:
     ✓ Cookie 及其他网站数据
     ✓ 缓存的图片和文件
  4. 点击"清除数据"
  ```

- [ ] **禁用广告拦截器**
  ```
  操作步骤:
  1. chrome://extensions/
  2. 找到广告拦截器（如 uBlock Origin）
  3. 关闭开关
  4. 或使用隐私模式（推荐）
  ```

---

### 3. Google 账号检查

- [ ] **Gmail 账号可用**
  - 测试: 访问 https://mail.google.com
  - 要求: 能正常登录
  - 注意: 使用个人账号（非企业账号）

- [ ] **手机号验证（可选）**
  ```
  如果要求验证:
  1. 选择"跳过"（如果可用）
  2. 或使用国外手机号
  3. 中国大陆手机号可能失败
  ```

- [ ] **账号状态正常**
  - 检查: https://myaccount.google.com
  - 确认: 无安全警告
  - 确认: 无使用限制

## 🛠️ 高级配置检查（可选）

### 6. API 配置

- [ ] **获取 API Key**
  ```
  步骤:
  1. 访问 https://ai.google.dev
  2. 点击"Get API key"
  3. 选择"Create API key in new project"
  4. 复制 API Key（格式: AIzaSy...）
  ```

- [ ] **测试 API 连接**
  ```bash
  curl -X POST \
    'https://generativelanguage.googleapis.com/v1beta/models/gemini-pro:generateContent?key=YOUR_API_KEY' \
    -H 'Content-Type: application/json' \
    -d '{
      "contents": [{
        "parts": [{
          "text": "Hello, Gemini!"
        }]
      }]
    }'
  ```

- [ ] **安装 Python SDK**（如需编程）
  ```bash
  pip install google-generativeai
  ```
