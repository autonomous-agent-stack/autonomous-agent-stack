# 🚀 Gemini 快速上手指南（5 分钟版）

> **目标**: 5 分钟内成功访问 Gemini  
> **最后更新**: 2026-03-30

---

## ⚡ 3 步快速开始

### 第 1 步: 开启 VPN（必需）

```
✅ 连接到美国/日本/新加坡节点
✅ 测试: 访问 https://www.google.com
```

### 第 2 步: 设置 Chrome 语言为英语（必需）

```
1. Chrome 设置 (chrome://settings/)
2. 左侧 → 语言
3. 添加语言 → English (United States)
4. 右侧 ⋮ → 移到顶部
5. 重启浏览器
```

### 第 3 步: 访问 Gemini

```
1. 打开 https://gemini.google.com
2. 点击右上角"Sign in"
3. 输入 Gmail 和密码
4. 开始使用！
```

---

## 🔥 核心要点

### ✅ 必须做

- **开启 VPN**: 否则无法访问
- **浏览器语言设为英语**: 避免区域限制
- **使用 Gmail 登录**: Google 账号必需

### ❌ 不要做

- 不要使用中文浏览器界面
- 不要关闭 VPN（使用期间全程开启）
- 不要使用中国大陆手机号验证（可选）

---

## 🛠️ 故障排除（快速版）

| 问题 | 解决方案 |
|------|----------|
| 无法访问 | 检查 VPN → 切换美国节点 → 清除缓存 |
| 一直转圈 | 更换节点 → 关闭广告拦截器 |
| 登录失败 | 检查账号 → 使用隐私模式 |

2. 下载并安装
3. 开启 VPN
4. 打开 App → 登录 Google 账号

```

```
