## 🔥 Claude Code 相关课程（高优先级）

### 1. Claude Code in Action ⭐⭐⭐
- **链接**: https://anthropic.skilljar.com/claude-code-in-action
- **时长**: 1 小时视频 + 15 讲座 + 1 测验
- **证书**: ✅ 完成证书
- **价格**: FREE

**学习目标**:
- 使用 Claude Code 核心工具（文件操作、命令执行、代码分析）
- 上下文管理（`/init`、CLAUDE.md、`@` 提及）
- 对话流程控制（热键和命令）
- 启用 Plan Mode 和 Thinking Mode
- 创建自定义命令自动化重复工作流
- 扩展 Claude Code（MCP servers、浏览器自动化）
- 设置 GitHub 集成（PR reviews、issue handling）
- 编写 hooks 添加额外行为

**适合人群**: 想用 AI 加速开发流程的工程师

---

### 2. Introduction to Agent Skills ⭐⭐⭐
- **链接**: https://anthropic.skilljar.com/introduction-to-agent-skills
- **价格**: FREE

**课程内容**:
- 什么是 Skills（与 CLAUDE.md、hooks、subagents 的区别）
- 创建第一个 Skill（SKILL.md frontmatter、描述、目录组织）
- 配置和多文件 skills
- Skills vs. 其他 Claude Code 功能
- 分享 Skills（团队仓库、plugins、企业设置）
- Troubleshooting skills

**适合人群**: 想构建可复用 Claude Code 工作流的开发者

---

### 3. Introduction to Subagents ⭐⭐⭐
- **链接**: https://anthropic.skilljar.com/introduction-to-subagents
- **价格**: FREE

**学习内容**:
- Subagents 如何工作（独立上下文窗口、输入输出流）
- 创建自定义 subagents（`/agents` 命令）
- 设计有效的 subagents（结构化输出、障碍报告、工具访问限制）
- 何时使用（何时不使用）subagents

**适合人群**: 需要管理复杂任务、保持主对话清晰的开发者
