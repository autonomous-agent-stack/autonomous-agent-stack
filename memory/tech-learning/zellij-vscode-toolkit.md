## 💡 **与 Zellij 的关系**

### **Zellij 是什么？**
- **终端复用器**: 类似 tmux/screen
- **Rust 编写**: 高性能
- **现代设计**: 用户友好
- **配置语言**: KDL (KDL Document Language)

### **痛点解决**
- ❌ **配置复杂**: KDL 语法不熟悉
- ❌ **无提示**: 容易写错
- ❌ **无验证**: 运行时才发现错误
- ✅ **Zellij Toolkit**: 解决所有痛点

## 🔧 **安装方法**

### **VS Code 扩展市场**
```bash
# 搜索扩展
code --install-extension atoolz.zellij-vscode-toolkit
```

### **从源码安装**
```bash
git clone https://github.com/atoolz/zellij-vscode-toolkit.git
cd zellij-vscode-toolkit
npm install
npm run compile
```
