## 💡 经验教训

### 1. 模块路径验证
- **教训**: 不要假设模块路径正确
- **解决**: 总是先用 `python -c` 测试导入
- **工具**: 创建 `test_fastapi.sh` 自动验证

### 2. 端点地址规范
- **教训**: `/admin` 不等于 `/api/v1/admin/view`
- **解决**: 检查 OpenAPI 规范确认正确路径
- **工具**: `curl -s http://127.0.0.1:8001/openapi.json | jq '.paths | keys'`

### 3. Git 推送策略
- **教训**: 先 fetch 再 push，避免 non-fast-forward 错误
- **解决**: `git pull --rebase origin main` 然后 `git push`
- **工具**: 使用 rebase 而不是 merge
