## ⚙️ 配置规则

### 1. 模型选择原则

```
1. 优先使用高并发模型（Search-Std 50, Z1系列 60）
2. 根据任务复杂度选择模型等级
3. 避免使用低并发模型处理批量任务
4. 视觉任务优先 GLM-4.5V（10 并发）
```

### 2. 并发分配原则

```
1. 单个任务不超过总并发数的 30%
2. 保留 10% 并发作为缓冲
3. 高优先级任务预留专用并发
4. 监控系统资源使用率
```

### 3. 避免使用

```
❌ GLM-5 系列（除非用户明确要求）
❌ GLM-4.6（除非用户明确要求）
❌ GLM-4.7（除非用户明确要求）
```
