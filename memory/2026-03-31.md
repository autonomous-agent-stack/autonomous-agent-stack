### 🎯 关键决策

#### 1. 愿景调整
- **不是**: 18 天做出"无需人类干预、还能自我进化的超级智能体"
- **而是**: 18 天做出"能跑的控制平面 + 2-3 个 worker + 1 条闭环业务流"

#### 2. 三层架构
- **私人层**: 你的 OpenClaw 管家（私人前台）
- **控制层**: 你的调度中心（后台总控）
- **执行层**: OpenClaw + Linux + Mac + Win+影刀

#### 3. 核心原则
- **它调度 OpenClaw，不替代 OpenClaw**
- **让项目当 control plane，让 OpenClaw 当成熟 agent worker**

#### 4. 专业 Agent 工厂
- **不是**: 一个万能影刀 agent
- **而是**: 一个能批量生产、分发、治理专业 agent 的 control plane

---

### 📦 专业 Agent 模板（5 个）

1. **表单录入 Agent** - 把标准化数据喂给影刀流程
2. **对账 Agent** - Linux 清洗数据 + Win+影刀回填
3. **店铺巡检 Agent** - OpenClaw/浏览器抓数 + 影刀补全
4. **报表 Agent** - 抓取、清洗、生成、投递一条龙
5. **客服工单 Agent** - 分类、摘要、分流、后台操作
