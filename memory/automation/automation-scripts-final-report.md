## 🚀 使用方式

### 快速启动（推荐）

```bash
# 使用管理器
~/.openclaw/scripts/automation-manager.sh

# 或创建别名
alias autoscript='~/.openclaw/scripts/automation-manager.sh'
```

### 直接运行

```bash
# GitHub 备份
~/.openclaw/scripts/github-auto-backup.sh <repo-url>

# 日报生成
~/.openclaw/scripts/daily-report-generator.sh

# 代码检查
~/.openclaw/scripts/code-quality-check.sh
```

### 集成到工作流

```bash
# 添加到 PATH
export PATH="$HOME/.openclaw/scripts:$PATH"

# 创建每日工作流
~/.openclaw/scripts/daily-workflow.sh
```
