## 🎓 技术总结

### 核心流程

```
用户提问
   ↓
大模型识别意图
   ↓
提取参数（platform）
   ↓
调用本地 Skill
   ↓
生成平台专属文案
   ↓
返回给大模型
   ↓
生成自然语言回复
```

### 关键技术

1. **Function Calling** - 工具调用机制
2. **LiteLLM** - 统一 API 接口
3. **本地 Skill** - 稳定可靠的执行
