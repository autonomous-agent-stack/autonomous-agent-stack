### GitHub 深度审查工作流

**执行链路**:
```
┌─────────────────┐
│ GitHub Analyzer │  Node 1: 技能执行
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│   Claude CLI    │  Node 2: 架构分析
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  情报组装      │  Node 3: 数据组装
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│ #市场情报       │  Node 4: 投递目标
└─────────────────┘
```

**节点详情**:

| 节点 | 职责 | 执行者 | 耗时 |
|------|------|--------|------|
| **Node 1** | 执行 github-analyzer | 演化专家 Agent | ~1s |
| **Node 2** | Claude CLI 架构分析 | 架构总师 Agent | ~2s |
| **Node 3** | 组装情报包 | 通信网关 Agent | ~0.1s |
| **Node 4** | 投递到 #市场情报 | 通信网关 Agent | ~0.1s |
