## 💡 使用技巧

### 1. 选择合适的场景

根据你的当前任务选择对应的场景：

| 任务类型 | 推荐场景 |
|---------|---------|
| 新项目开发 | 代码生成 → 从零生成完整项目 |
| API 开发 | 代码生成 → REST API 端点 |
| 性能优化 | 代码优化 → 算法优化、内存优化 |
| Bug 调试 | Debug 技巧 → 日志调试、异常跟踪 |
| 代码审查 | 代码审查 → 安全漏洞检查 |
| 代码重构 | 代码重构 → 提取方法、引入设计模式 |

### 2. 定制 Prompt 模板

每个场景都提供了 Prompt 模板，你可以：

- 填写你的具体需求
- 添加项目特定的约束
- 调整输出格式
- 添加示例输入和输出

**示例**：
```
创建一个 REST API 项目，要求：
- 技术栈：Python + FastAPI + SQLAlchemy
- 核心功能：用户管理、文章发布、评论系统
- 项目结构：遵循 MVC 最佳实践
- 包含：README.md、配置文件、示例代码、pytest 测试框架
- 代码规范：遵循 PEP 8
```

### 3. 结合代码示例

- 先看代码示例，理解实现方式
- 再使用 Prompt 模板，生成定制化代码
- 参考最佳实践，优化代码质量
