## 🔥 官方文档引用

### Claude Code核心概念
Claude Code 是一个 AI 驱动的编程助手，帮助你：

1. **编写代码** - 自然语言描述需求，自动生成代码
2. **调试代码** - 智能识别和修复错误
3. **优化代码** - 提供性能优化建议
4. **学习编程** - 边做边学，实时指导

### 快速开始
```bash
# 安装Claude Code
npm install -g @anthropic/claude-code

# 登录
claude auth login

# 初始化项目
claude init

# 开始编程
claude chat
```
