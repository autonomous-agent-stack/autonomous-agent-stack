# Claude Code 非技术人员30天实战训练营 🚀

> 从零开始，30天成为Claude Code实战高手

## 📚 训练营介绍

这是一个专为非技术人员设计的30天Claude Code实战训练营。通过系统性学习和9个实战项目，让你从零基础快速掌握AI辅助编程的核心技能。

## 🎯 学习目标

- 掌握Claude Code的基本使用方法
- 学会使用AI辅助编写、调试、优化代码
- 完成9个实战项目，积累实际经验
- 建立良好的编程思维和代码规范
- 掌握Git版本控制基础

## 📖 如何使用本训练营

### 第1步：阅读课程大纲
先查看 `01-30-day-curriculum.md`，了解整体学习路径和每天的学习目标。

### 第2步：开始学习
按照课程大纲的顺序，每天完成对应的学习内容和练习。

### 第3步：实战项目
每周完成指定的实战项目，从模板开始，逐步完善功能。

### 第4步：巩固练习
完成每周的习题，检验学习成果。

### 第5步：记录笔记
使用每日笔记模板记录学习心得、遇到的问题和解决方案。
