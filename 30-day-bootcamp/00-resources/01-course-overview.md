## 🚀 快速开始

### 第1步：环境准备
```bash
# 安装Python
brew install python3

# 安装Claude Code
npm install -g @anthropic/claude-code

# 验证安装
python3 --version
claude --version
```

### 第2步：创建学习目录
```bash
mkdir ~/claude-code-bootcamp
cd ~/claude-code-bootcamp
```

### 第3步：开始第一天
```bash
# 查看Day 1课程
cat 01-week-1/Day-01-Environment-Setup.md

# 开始学习！
```

## 💡 学习技巧

### 高效学习方法
1. **项目驱动** - 边学边做项目
2. **AI辅助** - 充分利用Claude Code
3. **及时提问** - 遇到问题不拖延
